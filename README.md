# Simulación Electoral Perú

Este proyecto construye una simulación del resultado electoral en Perú utilizando datos públicos de la ONPE.

## Objetivo

Explorar escenarios donde la distribución del voto restante podría alterar el orden de los candidatos.

## Metodología

- Scraping de resultados oficiales
- Extrapolación por nivel provincial
- Análisis del voto restante

## Disclaimer

Este es un ejercicio académico. No representa una predicción oficial.
